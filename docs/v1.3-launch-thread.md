# SourceCapsule v1.3 launch thread

## Recommendation

Launch with a four-post English X thread after the Chrome Web Store listing is live. Use the three
store screenshots as proof, one per post. Publish the Chinese version as a quote-thread 12–24
hours later so each language stays readable without turning the main launch into an eight-post
bilingual wall.

Do not use a single post: v1.3 has too much to prove and the receipt/offline-output screenshots are
the strongest credibility assets. Do not lead with a long-form post or X Article: both add reading
friction before users understand the product. A separate Article can become evergreen educational
content later, but it is not the launch vehicle.

Replace `[CHROME WEB STORE URL]` only after the listing is approved. Do not announce Store
availability while the listing is still under review.

## 简体中文发布串文

### 1/4 — 配图 1

X 的书签只是入口，不是真正的存档。原帖被删、被改或沉底后，书签也救不了你。

SourceCapsule v1.3 可以把单帖、完整串文或 Article 保存成干净的 Markdown 和可离线阅读的
HTML，并尽量保留图片、投票、引用帖和视频。

免费、开源。↓

### 2/4 — 配图 2

这次我最在意的改进，是“诚实的保存回执”。

SourceCapsule 会清楚列出保存了多少帖子、投票、图片和视频；如果有内容无法完整保存，也会
直接告诉你，不会假装一切完整。

### 3/4 — 配图 3

导出的内容断网后依然能看。

你会得到可搜索、可阅读的 HTML，以及适合交给 ChatGPT、Claude 或个人知识库的 Markdown
和真实媒体文件。普通保存只留在本机；限时分享链接完全可选。

### 4/4

v1.3 还改进了串文识别、投票前后状态保存、长文 Note 恢复，并加入了正式的 Chrome
扩展界面。
